Herencia de Estilos CSS

INHERITANCE

En CSS, los elementos hijos heredan propiedades de sus padres como color, font-family, font-size y line-height. Aquí se muestra cómo funciona en la práctica.

Ejemplo 1 — Herencia de tipografía

🔵 Elemento padre — define: font-family, color, line-height

↳ Hijo 1 — hereda todo del padre

↳ ↳ Nieto — sigue heredando aunque esté 2 niveles abajo

↳ Hijo 2 — hereda pero sobreescribe el color

Ejemplo 2 — Herencia en cascada

El contenedor padre define font-family y color. Todas las tarjetas hijas los heredan automáticamente sin necesidad de repetirlos.

Ejemplo 3 — ¿Qué propiedades se heredan?

Propiedad CSS ¿Se hereda? Ejemplo de uso
color ✅ Sí El color del texto se hereda a todos los hijos
font-family ✅ Sí La fuente definida en body aplica a toda la página
font-size ✅ Sí Los hijos pueden usar em/rem relativos al padre
line-height ✅ Sí El interlineado se propaga a los elementos hijos
text-align ✅ Sí La alineación del texto se hereda al contenido
background ❌ No Cada elemento define su propio fondo
margin ❌ No Los márgenes no se heredan, son del elemento
padding ❌ No El relleno tampoco se hereda entre elementos
border ❌ No Los bordes son exclusivos de cada elemento
width / height ❌ No Las dimensiones no se heredan automáticamente
