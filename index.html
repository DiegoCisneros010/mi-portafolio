Resumen de propiedades Flexbox

Propiedad Se aplica en Descripción
display: flex Contenedor Activa el modo Flexbox en el contenedor
flex-direction Contenedor Define la dirección del eje principal (row / column)
justify-content Contenedor Alineación en el eje principal
align-items Contenedor Alineación en el eje secundario
flex-wrap Contenedor Permite que los elementos se envuelvan en varias filas
gap Contenedor Espacio entre elementos hijos
flex-grow Hijo Cuánto crece un hijo para llenar el espacio disponible
flex-shrink Hijo Cuánto se encoge un hijo si no hay suficiente espacio
align-self Hijo Alineación individual que sobreescribe align-items
order Hijo Controla el orden visual de un elemento hijo
